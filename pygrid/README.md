## Check if pygrid is running

```bash
ps aux | grep pygrid
```
## Kill that process

```bash
kill <procces num>
```

## Run binary in pygrid's repo

```bash
./src/bin/x11pygrid
```

## Add to ~/.profile

```bash
/home/julian/utils/x11pygrid/src/bin/x11pygrid &
```
## Reboot

```bash
sudo reboot
```
## Modify pygrid's shortcuts

```bash
cd ~/.config
code x11pygrid.json
```
